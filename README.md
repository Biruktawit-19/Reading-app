# DailyPage 📚

DailyPage is a web application designed to help users build a daily reading habit through goal tracking, progress monitoring, and interactive reading features.

## Features

* User account system (signup/login)
* Daily reading goals
* Reading progress tracker
* Reading streak system
* Book library with categories
* Interactive book reader
* Audiobook support for accessibility

## Technologies Used

* HTML
* CSS
* JavaScript
* Firebase (authentication and database)

## Purpose

The goal of this project is to encourage people to develop a consistent reading habit and make reading accessible for everyone, including visually impaired users.
